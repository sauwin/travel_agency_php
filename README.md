# Travel Agency PHP Project
This is a simple travel agency web application built with PHP.  
It allows managing tours, users, and reservations through a web interface.  

The project is implemented using **Object-Oriented Programming (OOP)** and supports **CRUD operations** for main entities.  

## Features
- Add, update, delete, and list tours
- User registration and management
- Reservation management
- Data stored in database MySQL

## Requirements
- PHP 7.4+  
- Web server Apache
- Browser for accessing the web interface
